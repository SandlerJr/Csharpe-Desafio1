# Projeto de Prática em Programação

Este projeto foi criado com o objetivo de praticar conceitos básicos de programação, como variáveis, listas e entrada de dados. As atividades não são obrigatórias, mas ajudam a consolidar o aprendizado.

## 🚀 Atividades

1. **Variável de nota média**
   - Criar uma variável chamada `notaMedia` e atribuir um valor inteiro a ela.
   - Caso o valor seja maior ou igual a 5, exibir na tela:  
     ```
     Nota suficiente para aprovação
     ```

2. **Lista de linguagens de programação**
   - Criar uma lista contendo as linguagens:  
     - C#  
     - Java  
     - JavaScript  

3. **Exibir valor da lista**
   - Utilizar a lista criada no exercício anterior para exibir o valor `"C#"` no console.

4. **Entrada de dados e posição na lista**
   - Criar um programa que receba um número inteiro `n` digitado pelo usuário.
   - Exibir o elemento da lista que está na posição `n`.

## 🎯 Objetivo

O objetivo é praticar:
- Declaração e uso de variáveis
